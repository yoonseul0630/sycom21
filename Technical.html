      이는 DIF (Design for Intergration) 라는 디자인 철학입니다.
      또한 설계 단계에서 안테나를 쉽게 적용할 수 있도록 필요에 따라 모든 안테나에 대한 CAD file을 무료롤 제공하고,
      아키텍처와 gerber file review를 무료로 제공합니다.
      • ● 디자인 초기 layout 지원
      • ● 안테나 CAD file 무료 제공
      • ● artwork 및 gerber file review 지원
